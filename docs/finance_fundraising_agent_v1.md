# Finance-Fundraising Agent（财务融资）执行包 v1

## 角色目标

保障现金流安全，建立可融资叙事与数据证据。  
约束：优先 runway，不做激进扩张预算；融资叙事必须可被业务指标验证。

### 1.4 现金流安全规则（执行阈值）

- 一级预警：期末现金 < 450万（约3个月固定开销）  
  动作：冻结非关键招聘，营销预算下调30%，仅保交付/合规/核心研发。
- 二级预警：连续2个月净流出 > 80万  
  动作：暂停低毛利项目，收缩非标需求，回款条款前置。
- 预算解锁条件：基准回款达成率 > 85% 且毛利率 >= 55% 连续2个月成立。

## 2) 定价方案与敏感性分析

### 2.1 定价结构

1. 订阅制（主引擎，提升MRR质量）  
   - Pro：2.5万/月  
   - Enterprise：6.8万/月
2. 分成制（增量场景）  
   - 平台抽成：8%-15%  
   - 设置保底服务费：1万-3万/月
3. 私有化（高客单补充）  
   - 部署费：80万-300万  
   - 年维护：部署费的15%-20%

### 2.2 敏感性分析（基准）

| 变量变化 | MRR影响 | 毛利影响 | 结论 |
|---|---:|---:|---|
| 订阅单价 +10% | +7%-9% | +2-3pct | 可提价，需同步强化价值证明 |
| 订阅单价 -10% | -8%-12% | -2-4pct | 不建议，损害收入质量 |
| 分成比例 +3pct | +4%-6% | +1-2pct | 中等敏感，注意渠道博弈 |
| 私有化降价15% | +0~+3项目 | -5-8pct | 仅用于标杆签约，不可常态 |
| DSO拉长至120天 | 账面收入不变 | 现金恶化明显 | 优先优化回款条款 |

### 2.3 合同与报价红线

- 项目毛利率不得低于45%。  
- 回款节点优先于总额与单价。  
- 私有化折扣须绑定案例授权与复用资产（模板、可复制接口）。

## 3) 融资里程碑与关键指标

### 3.1 里程碑设计

| 阶段 | 时间 | 目标 | 验证指标 |
|---|---|---|---|
| Seed-Ready | M1-M3 | 证明闭环成立 | PoC >= 5，月回款 >= 50万，3月留存 >= 85% |
| Pre-A Story | M4-M6 | 证明可复制获客与交付 | MRR >= 120万，付费客户 >= 12，毛利率 >= 55% |
| Raise Window | M6-M8 | 发起并完成融资 | MRR >= 180万，NRR >= 105%，DSO <= 90天 |
| Post-Raise | M9-M12 | 资金效率扩张 | CAC回收期 <= 12个月，连续2个月经营现金流转正 |

### 3.2 核心指标定义（周更）

- MRR：当月可持续经常性收入（剔除一次性部署）。  
- PoC数：签约且进入验收流程的PoC项目数量。  
- 留存：以客户数和收入双口径计算（Logo Retention/NRR）。  
- DSO：应收账款周转天数，作为现金流核心前置指标。
