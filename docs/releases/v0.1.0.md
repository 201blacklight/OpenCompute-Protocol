# Release v0.1.0

## Highlights
- Established a formal repository structure for strategy, architecture, governance, and demo delivery.
- Added bilingual documentation structure under `docs/zh/` and `docs/en/`.
- Added collaboration standards: `CONTRIBUTING.md`, PR template, and issue templates.
- Delivered runnable OCP demo with visual preview timeline and account delta tables.
- Included the original project idea archive under `docs/zh/idea/original_idea.md`.

## Included Scope
- API and service sources in `app/`
- API tests in `tests/`
- Demo engine and preview scripts in `demo/ocp_demo/` and `scripts/`
- Decision, architecture iteration, and governance documentation

## Validation
- Demo execution: `python demo/ocp_demo/main.py`
- Visual preview: `python demo/ocp_demo/visual_preview.py`
- Demo tests: `python -m pytest demo/ocp_demo/tests/test_engine.py -q`

## Notes
- This is the first baseline release for collaboration and iterative development.
