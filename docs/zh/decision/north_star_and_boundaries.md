# OCP 北极星目标与边界

## 北极星目标
- 构建 AI 支付、额度与结算互通的统一协议层。
- 在可审计规则下，把用户的现实购买力换算为协议 Token 额度。
- 先完成可落地互通，再逐步扩展到更广义的算力金融基础设施。

## 现实约束
- 初始阶段无资金、无人脉。
- 必须避免触碰资金托管与跨境清结算等监管红线。
- 必须先用开放标准与可复现 Demo 证明价值。

## 战略节奏
- Phase 0：协议草案 + 离线 Demo + 风险白皮书。
- Phase 1：支付沙箱与模型方沙箱接入。
- Phase 2：政府/行业试点与开放联盟标准化。

## 关键范围取舍
- 保留：统一 Token 化使用结算。
- 延后：完整 AI 所有权交易，先做收益分成权转移。
- 将“全局硬上限”改为分层额度上限：
  - 协议层额度
  - 厂商层额度
  - 用户层额度
